• 7 editores de código para Android

    Postagens > Dicas
    Publicado dia 21/02/2018 por AllyTiago

    Todos nós sabemos que para primeiro começar a codificar necessitamos conhecer sobre a determinada linguagem, e tambem sabemos que para mexer no código necessita de uma aplicação para poder ler e edita-lo. Sabendo que existem vários outros sites com conteúdos similares só que para os desktops, então eu trarei os melhores editores para o Android, atual sistema operacional da gigante Google.

    Primeiramente começando com algo basico partiremos pelo desenvolvimento web. Será apresentado 7 editores de códigos desenolvidos para funcionar no android.

    Os apps não estão em ordem do melhor de todos até o somente bom pois cada um deles possui funcões distintas ou até praticamente iguais então por isso foi decidido usar uma ordem aleatoria.


    1. AnWriter

    Editor de códigos simples, prático e funcional. Tem até essa data de publicação 100.000 downloads, possui ferramentas perfeitas para a edição como um suporte autocompletar para: HTML, CSS, JavaScript, LaTeX (PHP, SQL na versão Pro), de tags e atalhos no teclado, ftp, reconhecimento das classes e ids utilizadas e suporta teclado físico. Para poder efetuar o download clique aqui.


    2. Turbo Editor

    Contendo a mesma quantidade de downloads, o Turbo Editor é mais voltado aos tablets, sem autocomplete ou atalhos o unico diferencial é possuir um tema material, ser opensource ter o idioma português e possuir suporte para todos os formatos de ficheiro de texto mesmo que sejam grandes, função que a maioria limita até um determinado megabyte e possuir realce de sintaxe para HTML, XHTML, CSS, LESS, JS, PHP, PYTHON e muitas mais linguagens de programação detectar automáticamente a codificação e possibilidade de a altera-la. Para efetuar o download clique aqui.


    3. Dcoder, Mobile compiler IDE

    Iniciando com os compiladores, eu lhes apresento o Dcoder, que é uma ótima ferrameta para compilar até 31 linguagens dentre elas Java, C, C#, C++, Kotlin, Python, Lua, Php, Go, Ruby, NodeJs e outros. Possui uma comunidade para lhe auxiliar e disponibiliza a ferramenta Leader Board para saber onde você está entre a vasta comunidade Dcoder. Para efetuar o download clique aqui.


    4. Online Compiler

    Similar ao Dcoder, o Online Compiler é 100% gratuito e tem versão web, suporte a mais de 21 linguagens incluindo Pascal, MySQL, Bash, R, Scala, Swift e outros, ambiente Syntax Highlighting que tambem existe no outro app. Para efetuar o download clique aqui.

    Prefiro o Online Compiler ao invés do Dcoder por ser totalmente gratuito, ter aquilo que preciso, sem anúncios ou mensagens enjoadas oferençendo compras de produtos para o app.


    5. Quoda Code Editor

    Outra ferramenta similar ao AnWriter ganha destaque aqui na postagem, eu assumo que já utilizei as suas funções e me agradei, as diferenças aqui encontradas é o suporte ao Dropbox e ao FTP, conta com suporte às linguagens ActionScript, C, C++, C#, CSS, Haskell, HTML, Java, JavaScript, Lisp, Lua, Markdown, Objective-C, Perl, PHP, Python, Ruby, SQL, Visual Basic e XML, a função autocomplete nele fornecida é premium e só os que comprarem teram acesso e o mesmo ocorre no ftp, possui mais de um tema incluindo o famoso Monokay. Para efetuar o download clique aqui.


    6. Bluino Loader - Arduino IDE

    Iniciando na área de programação nos hardwares eu lhes apresento o Bluino Loader, um software para programação em arduino, com ele se torna fácil de escrever código e compilar esboço para gerar o arquivo hex e enviá-lo para placas Bluino ou várias outras placas Arduino via USB OTG ou Bluetooth sem fio. Para efetuar o download clique aqui.


    7. AIDE - IDE for Android Java C++

    O AIDE não é um único app mas sim um conjunto de aplicações para diversos tipos de programação. Sem fugir da temática abordaremos a IDE de desenvolvimento de aplicações Android com java ou c/c++. Neste app o usuário conta com um ambiente de desenvolvimento integrado para o desenvolvimento diretamente no seu dispositivo, e caso o usuário não entenda muito o proprio ambiente lhe forneçe cursos com várias lições de codificação interativa e passo-a-passo se tornar um desenvolvedor de aplicativo especialista, podendo tambem importar projetos do do Android Studio ou Eclipse e possuir integração com Git e Dropbox. Para efetuar o download clique aqui.
